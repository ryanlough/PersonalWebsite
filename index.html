Battle Blocks

Multiplayer Web Game

Written in C#, Battle Blocks is a fun multiplayer game that is run on top of the (then) popular game Everybody Edits. Through the use of the playerio and .NET frameworks, Battle Blocks receives/sends game-readable packets to interact with the online game that it is built on top of. To play, each of the six players would have a block they are able to control which leaves behind their respective color. The player who's color filled the most area of the board by the time the round was over wins.

Battle Blocks has over 30 different game boards, 10 powerups, nearly 10,000 plays across over 1000 players, and was completed in 2012.

  • Date: March 2012
  • Language: C#
  • Category: Multiplayer Gaming
